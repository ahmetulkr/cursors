# My Next.js App

Modern Next.js 15 projesi - TypeScript, Tailwind CSS ve App Router ile oluşturulmuştur.

## Başlangıç

Önce bağımlılıkları yükleyin:

```bash
npm install
# veya
yarn install
# veya
pnpm install
```

Daha sonra geliştirme sunucusunu başlatın:

```bash
npm run dev
# veya
yarn dev
# veya
pnpm dev
```

Tarayıcınızda [http://localhost:3000](http://localhost:3000) adresini açın.

## Proje Yapısı

- `src/app/` - App Router sayfaları ve layout'ları
- `src/app/page.tsx` - Ana sayfa
- `src/app/layout.tsx` - Root layout
- `src/app/globals.css` - Global stiller

## Özellikler

- ⚡ **Next.js 15** - En son Next.js sürümü
- ⚛️ **React 18** - Modern React özellikleri
- 🔷 **TypeScript** - Tip güvenliği
- 🎨 **Tailwind CSS** - Utility-first CSS framework
- 📱 **Responsive** - Mobil uyumlu tasarım
- 🌙 **Dark Mode** - Otomatik karanlık tema desteği

## Komutlar

- `npm run dev` - Geliştirme sunucusunu başlatır
- `npm run build` - Production için build alır
- `npm run start` - Production sunucusunu başlatır
- `npm run lint` - ESLint ile kod kontrolü yapar
